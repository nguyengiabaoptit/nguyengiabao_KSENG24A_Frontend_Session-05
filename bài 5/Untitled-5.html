5. Groups of selectors
    

    A comma-separated list of selectors represents the union of all elements

    selected by each of the individual selectors in the list. (A comma isU+002C.)

    For example, in CSS when several selectors share the samedeclarations, they 

    be grouped into a comma-separated list.
    


        CSS example:


        In this example, we condense three rules with identical declaration


        h1 { font-family: san-serif }

        h2 { font-family: san-serif }

        h3 { font-family: san-serif }

        

        is equivalent to:
        

        h1, h2, h3 { font-family: san-serif }
